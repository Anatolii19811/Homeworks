Блочна верстка

Блочна верстка — це тип дизайну сторінки, створений за допомогою блоків (тег

), розташування яких суворо контролюється через CSS (каскадних таблиць стилів), в якому таблиці використовуються тільки для представлення табличних даних. Перевагами даного типу є:

  • Кешування блоків браузером, за рахунок чого швидке завантаження сторінок;
  • Необмежені можливості по створенню складних дизайнів;
  • Чітке відділення частини з контентом;
  • Простота реалізації програмної частини;

Недоліком блокової верстки є складність реалізації відносно табличної для простих сайтів,також виникає багато проблем із забезпеченням кросбраузерності для старих браузерів, але при цьому вона відмінно індексується пошуковиками, дозволяє багатьом програмам, що працюють із контентом сайтів, правильно «читати» його вміст, допомагає користувачам швидко завантажувати сторінки,також за допомогою поєднання блоків CSS можна створювати професійні дизайни. Відповідно питання: чи можна створити професійний дизайн в табличній верстці? Звичайно можна: робиться це за блоками.

Блокова верстка з div

За допомогою розмітки визначається розташування елементів на вебсторінці. Наприклад, ми створюємо шапку сайту, всередині якої розміщуємо необхідний нам контент (логотип, номери телефонів, навігаційне меню і т. ін.).
